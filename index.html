  • Simple RPC server & client [Github Description Document]

    This is a C++ project for a distribution system course. It allows developers to set up custom functions to their server, so that users can run these functions in the "cloud". It has 3 major parts.

    • The coordinator server acts as a gateway between servers and clients. Mainly, it provides clients namespace resolution of the other servers. It also acts as a worker server for computing purposes.
    • The worker servers provide cloud computing capabilities to the clients. These servers internally store a namespace directory that gets updated when they contact the coordinator.
    • The user uses the client application to request cloud computation and to get the results. The client stores a name directory, and it synchronizes its name directory with a worker server under successful contact, allowing local name lookup; hence lessening the loads of the coordinator.

    Note: This project needs to be compiled and run in a Linux environment.
